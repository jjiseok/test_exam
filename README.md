# 중학교 시험감독 배정표 자동 생성 시스템

중학교 정기고사 시험감독을 공정하게 자동 배정하고, 교사별 정감독·부감독·복도감독 횟수를 연간 누적 관리하는 웹앱입니다.

## 1. 포함 기능

- 교사 명단 엑셀 업로드
- 교사 추가, 수정, 삭제
- 감독 가능 여부 및 감독 유형별 제외 설정
- 특정 날짜 제외, 특정 교시 제외
- 시험 정보 생성: 학년도, 학기, 1차/2차, 시험명
- 시험 시간표 입력: 시험일자, 교시, 시작/종료 시간, 학년, 교과, 시험실 수
- 시험 시간표 엑셀 업로드
- 해당 시간 시험 교과 담당 교사 복도감독 우선 배정
- 같은 날짜·교시 교사 중복 배정 방지
- 누적 감독 횟수가 적은 교사 우선 배정
- 같은 조건일 경우 랜덤 배정
- 재배정 가능
- 배정 결과 수동 수정 가능
- 교사별 연간 감독 통계 자동 계산
- 엑셀 다운로드
- 인쇄 또는 PDF 저장용 화면

## 2. 실행 방법

### Docker Compose로 실행

```bash
docker-compose up --build
```

실행 후 접속합니다.

- 웹 화면: http://localhost:5173
- API 문서: http://localhost:8000/docs

### 로컬 실행

#### 백엔드

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

#### 프론트엔드

```bash
cd frontend
npm install
npm run dev
```

## 3. 기본 사용 순서

1. `예시 데이터 만들기`를 눌러 기능을 먼저 확인합니다.
2. `3. 자동 배정` 탭에서 `랜덤 자동 배정 실행`을 누릅니다.
3. 결과표에서 배정 결과를 확인합니다.
4. 필요한 경우 교사 선택 상자를 바꾸어 수동 수정합니다.
5. `4. 누적 통계`에서 교사별 정감독·부감독·복도감독 횟수를 확인합니다.
6. `엑셀 다운로드` 또는 `5. 인쇄/PDF`에서 출력합니다.

실제 사용 시에는 다음 순서로 진행합니다.

1. 교사 엑셀 양식 다운로드
2. 교사 명단 작성 후 업로드
3. 시험 정보 생성
4. 시험 시간표 직접 입력 또는 엑셀 업로드
5. 자동 배정 실행
6. 결과 확인 및 수정
7. 엑셀 다운로드 또는 PDF 저장

## 4. 교사 명단 엑셀 양식

다음 열을 사용합니다.

| 열 이름 | 설명 |
|---|---|
| 교사명 | 필수 |
| 교과 | 국어, 수학, 영어 등 |
| 담임 여부 | 예/아니오 |
| 담당 학년 | 1, 2, 3 등 |
| 담당 반 | 1, 2, 3 등 |
| 부서 | 교무기획부 등 |
| 감독 가능 여부 | 예/아니오 |
| 정감독 제외 여부 | 예/아니오 |
| 부감독 제외 여부 | 예/아니오 |
| 복도감독 제외 여부 | 예/아니오 |
| 특정 날짜 제외 | 예: 2026-04-30,2026-05-01 |
| 특정 교시 제외 | 예: 1,3 또는 1교시,3교시 |
| 비고 | 참고사항 |

웹 화면에서 `교사 엑셀 양식 다운로드`를 누르면 바로 양식을 받을 수 있습니다.

## 5. 시험 시간표 엑셀 양식

다음 열을 사용합니다.

| 열 이름 | 설명 |
|---|---|
| 시험일자 | 예: 2026-04-30 |
| 교시 | 예: 1 |

| 종료시간 | 예: 09:45 |

웹 화면에서 `시험 시간표 양식 다운로드`를 누르면 바로 양식을 받을 수 있습니다.

## 6. 자동 배정 알고리즘

자동 배정은 다음 순서로 진행됩니다.

1. 시험 일정과 교시 정보를 불러옵니다.
2. 교사 명단과 제외 조건을 불러옵니다.
3. 해당 교시에 시험이 있는 교과 담당 교사를 확인합니다.
4. 교과 담당 교사를 복도감독 후보로 우선 분류합니다.
5. 같은 날짜·교시에 이미 배정된 교사는 후보에서 제외합니다.
6. 전체 감독 제외, 정감독 제외, 부감독 제외, 복도감독 제외, 특정 날짜 제외, 특정 교시 제외 조건을 반영합니다.
7. 복도감독을 먼저 배정합니다.
8. 시험실별 정감독과 부감독을 배정합니다.
9. 연간 누적 감독 횟수와 현재 시험 기간의 감독 횟수가 적은 교사를 우선합니다.
10. 같은 조건에서는 랜덤으로 배정합니다.
11. 배정 결과를 저장하고 통계를 다시 계산합니다.

## 7. 데이터베이스 구조

SQLite 파일은 `backend/data/exam_supervision.db`에 저장됩니다.

주요 테이블은 다음과 같습니다.

- `teachers`: 교사 명단 및 제외 조건
- `exams`: 시험 기본 정보
- `exam_slots`: 시험일자, 교시, 시간, 학년, 교과, 시험실 수
- `assignments`: 정감독, 부감독, 복도감독 배정 결과

## 8. 주의 사항

- 교과명은 교사 명단의 `교과`와 시험 시간표의 `교과`가 정확히 일치해야 복도감독 우선 배정이 잘 작동합니다.
- 특정 날짜 제외는 `YYYY-MM-DD` 형식으로 입력하는 것이 좋습니다.
- 특정 교시 제외는 `1,3` 또는 `1교시,3교시`처럼 입력할 수 있습니다.
- 교사 수가 부족하면 `미배정`으로 표시되며 경고 메시지가 나타납니다.
- 인쇄 화면에서 브라우저의 `PDF로 저장` 기능을 사용하면 PDF 파일로 저장할 수 있습니다.
